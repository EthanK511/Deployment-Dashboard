Token needs: repo and pages permissions

To generate a token, follow the instructions below

1. Navigate via GitHub to github.com/settings/personal-access-tokens

2. Click "Generate New Token"

3. Name the token something like "Deployment Dashboard"

4. Select All Repositories under Repository Access

5. Click Add Permissions under the permissions tab

6. Select Deployments and Pages and make sure they are set to read and write

7. Press save or update and copy the code to the field above
